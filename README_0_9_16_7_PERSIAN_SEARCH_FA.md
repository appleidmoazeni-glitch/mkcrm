# MKCRM 0.9.16.7 - اصلاح جستجوی فارسی/عربی

## اصلاحات

- نرمال‌سازی حروف فارسی/عربی در جستجوی SQL شایگان اضافه شد.
- مشکل تفاوت `ی / ي` و `ک / ك` در سرچ کالا، موجودی، فاکتور فروش، کالاها، حساب‌ها و تأمین‌کننده‌ها اصلاح شد.
- دیتای شایگان تغییر نمی‌کند؛ فقط لایه مقایسه و جستجو نرمال می‌شود.
- اتصال نرم Lead ID و اصلاح dedupe گزارش Lead از نسخه قبل حفظ شده است.

## تست پیشنهادی

در CRM این عبارت‌ها را در فاکتور فروش و موجودی انبارها تست کنید:

- جیهون
- جيهون
- لیبل کیبورد
- ليبل كيبورد
- کیبورد
- كيبورد

نتیجه باید برای شکل فارسی و عربی یکسان باشد.
