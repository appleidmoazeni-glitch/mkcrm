# MKCRM 0.9.16.0 - Manual Lead ID Soft Link

این نسخه اتصال زنده به برنامه Lead ID ایجاد نمی‌کند. فروشنده/Closer کد Lead ID را به‌صورت دستی در فاکتور فروش وارد می‌کند.

## تغییرات
- Lead ID در فاکتور فروش دستی باقی ماند.
- فاکتور بدون Lead ID در دوره آزمایشی مجاز است، اما با `invoiceWithoutLead=true` و `leadPenaltyEligible=true` در Mongo ثبت می‌شود.
- Lead ID در توضیحات فاکتور شایگان ذخیره می‌شود؛ اگر خالی باشد توضیح اضافه نمی‌شود.
- گزارش API برای کنترل فاکتورهای بدون Lead اضافه شد: `/api/reports/sale-lead-audit?days=31`.
- قفل صدور همزمان فاکتور و جلوگیری از ثبت تکراری حفظ شد.

## تست
1. با فروشنده فاکتور با Lead ID ثبت کن و توضیحات فاکتور شایگان را بررسی کن.
2. با فروشنده فاکتور بدون Lead ID ثبت کن؛ باید ثبت شود ولی در گزارش audit به‌عنوان بدون لید بیاید.
3. همزمان دوبار دکمه صدور را بزن؛ باید قفل جلوگیری از تکرار فعال بماند.
