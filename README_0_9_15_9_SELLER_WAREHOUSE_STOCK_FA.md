# MKCRM 0.9.15.9 - دسترسی فروشنده به موجودی کامل انبار انتخاب‌شده

این نسخه بر پایه 0.9.15.8 ساخته شده است.

## اصلاح
- فروشنده و فروشنده-خریدار هم می‌توانند در منوی «موجودی انبارها» یک انبار انتخاب کنند و دکمه «نمایش کل موجودی انبار انتخاب‌شده» را بزنند.
- خروجی فقط موجودی مثبت همان انبار را از SQL مستقیم شایگان می‌خواند.
- نمایش «همه انبارها» همچنان برای جلوگیری از سنگین شدن صفحه غیرفعال است.

## تست
1. با اکانت فروشنده وارد شوید.
2. منوی موجودی انبارها را باز کنید.
3. یک انبار مشخص انتخاب کنید.
4. دکمه نمایش کل موجودی انبار انتخاب‌شده را بزنید.
5. باید ردیف‌های دارای موجودی همان انبار نمایش داده شود.
