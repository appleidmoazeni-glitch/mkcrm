# MKCRM 0.9.17.2 - Customer Channel Filter Fix

اصلاح فیلتر مشتریان سایت و مشتریان کنسول در بانک مشتریان.

## تغییرات
- فیلتر مشتریان سایت فقط به salesChannel وابسته نیست و از نام صندوق/فروشگاه/منبع هم تشخیص می‌دهد.
- فیلتر مشتریان کنسول فقط به businessUnit وابسته نیست و از نام صندوق کنسول هم تشخیص می‌دهد.
- endpoint جدید برای اصلاح برچسب‌های کانال روی داده‌های قبلی اضافه شد:
  POST /api/customers/repair-channel-tags

## بعد از نصب
از داخل ادمین یا با curl این عملیات را یک بار اجرا کنید تا داده‌های sync شده قبلی هم برچسب‌گذاری شوند.

.env عملیاتی را overwrite نکن. تنظیمات Lead ID باید باقی بمانند:
LEAD_ID_BASE_URL=http://127.0.0.1:8010
LEAD_ID_API_KEY=
LEAD_ID_TIMEOUT_MS=2500
