# MKCRM 0.9.19.11 - فیلتر نوت‌بوک عملکرد فروشنده

- فیلتر گروه کالایی نوت‌بوک در صفحه عملکرد فروشنده اضافه شد.
- منطق فیلتر: اگر کد کالا با رقم `1` شروع شود، گروه اصلی نوت‌بوک محسوب می‌شود.
- فیلتر روی ردیف‌های فاکتور اعمال می‌شود؛ در فاکتورهای ترکیبی فقط ردیف‌های نوت‌بوک وارد فروش/سود FIFO می‌شوند.
- منبع فروش همچنان WebService فاکتورهای فروش شایگان است و سود با FIFO SQL محاسبه می‌شود.
