# MKCRM 0.9.19.14 - اصلاح تاریخ شمسی عملکرد فروشنده

- فیلتر تاریخ عملکرد فروشنده در UI شمسی شد.
- مقدار پیش‌فرض از ابتدای ماه شمسی جاری تا امروز شمسی است.
- ورودی قابل قبول: `14050301` یا `1405/03/01` یا `1405-03-01`.
- سرور قبل از خواندن فاکتورهای فروش از WebService شایگان، تاریخ شمسی را به فرمت قابل قبول شایگان `YYYYMMDD` تبدیل می‌کند.
- فیلترهای سود و نوت‌بوک همچنان فقط روی cache آخرین گزارش اعمال می‌شوند و باعث محاسبه مجدد نمی‌شوند.
