# MKCRM 0.9.16.1 - Lead Audit Report + Supplier Layer Filter

## اصلاحات

1. فعال‌سازی و مقاوم‌سازی گزارش کنترل Lead ID فاکتور فروش:
   - `/api/reports/sale-lead-audit?days=31`
   - خواندن از `saleIssueLocks`
   - fallback به `invoiceAuditLogs` برای فاکتورهای قبلی/ساختار قدیمی
   - نمایش فاکتورهای دارای Lead ID و بدون Lead ID
   - نمایش وضعیت مشمول کسر پورسانت

2. افزودن فیلتر تأمین‌کننده در گزارش خواب کالا - بخش لایه‌های کالایی:
   - انتخاب تأمین‌کننده از لیست تأمین‌کننده‌های فعال
   - محاسبه backend با `supplierAccountId`
   - نمایش فقط موجودی منتسب به همان تأمین‌کننده
   - خلاصه ارزش موجودی، تعداد لایه، میانگین خواب فیزیکی و میانگین فشار مالی برای همان تأمین‌کننده

## تست

### گزارش Lead ID

```bat
curl http://127.0.0.1:1385/api/reports/sale-lead-audit?days=31
```

با لاگین ادمین در مرورگر هم قابل تست است.

### گزارش خواب کالا

1. ورود با ادمین
2. منوی خواب کالا - تأمین‌کننده
3. انتخاب تأمین‌کننده در فیلتر «لایه‌های کالایی»
4. محاسبه گزارش
5. کنترل اینکه فقط کالاهای همان تأمین‌کننده نمایش داده شود
