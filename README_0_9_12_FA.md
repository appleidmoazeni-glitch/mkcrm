# MKCRM 0.9.12

نسخه کامل بر پایه 0.9.11/0.9.10 با اصلاح یکپارچه موجودی و کاردکس.

قوانین اصلی:
- موجودی قابل فروش فقط live از شایگان معتبر است.
- کش فقط برای کاتالوگ/جستجوی کالای بدون موجودی استفاده می‌شود، نه برای فروش.
- کاردکس همه مسیرها از سرویس paging واحد خوانده می‌شود.
- پورت عملیاتی 1385 و PUBLIC_BASE_URL داخلی 172.31.31.217 حفظ شده است.

تست مرجع:
1. کالای بدون موجودی مثل M308 نباید در سرچ فروش قابل انتخاب با موجودی مثبت دیده شود.
2. بعد از انتخاب کالا در فروش، موجودی همان انبار دوباره live کنترل می‌شود.
3. کاردکس از منوی موجودی انبار باید popup بماند و با stockNumber همان ردیف باز شود.
4. کالای P10377702113 باید بیش از یک گردش را طبق سقف کاربر/ادمین نشان دهد.
