# MKCRM 0.9.19.43 - Seller Performance FIFO Profit Diagnostic

## تمرکز نسخه
- گزارش عملکرد نماینده فروش از Sale Snapshot واقعی InvTyp=2.
- فیلتر نماینده و تاریخ از Mongo، نه WebService مستقیم.
- محاسبه سود مدیریتی فروشنده با FIFO بر اساس Purchase Layerها.

## منطق سود
- همه فروش‌ها تا تاریخ پایان برای مصرف FIFO لحاظ می‌شوند؛ فقط فروش‌های نماینده/بازه انتخابی گزارش می‌شوند.
- هر SaleLine با ItemCode، تاریخ فروش و مقدار فروش به لایه‌های خرید همان کالا قبل از تاریخ فروش متصل می‌شود.
- اگر کالا با قیمت‌های مختلف خریداری شده باشد، FIFO از قدیمی‌ترین لایه واجد شرایط مصرف می‌کند.
- اگر لایه خرید کافی یا قابل اتصال وجود نداشته باشد سود صفر نمی‌شود؛ وضعیت `unknown` یا `partial` نمایش داده می‌شود.

## خروجی
- عدد فروش
- بهای FIFO
- سود FIFO
- ROI
- گروه اصلی کالا
- فاکتورها و ردیف‌ها
- Diagnostic ردیف‌های محاسبه‌شده/ناقص/نامشخص

## نکته
سود این نسخه مدیریتی/قابل ردیابی است و برای نهایی شدن باید با چند فاکتور نمونه و قیمت خرید شایگان کنترل شود.
