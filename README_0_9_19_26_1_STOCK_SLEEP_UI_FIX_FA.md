# MKCRM 0.9.19.26.1 - Stock Sleep UI Fix

این نسخه اصلاحی برای 0.9.19.26 است.

## مشکل نسخه قبلی
- موتور backend خواب کالا اضافه شده بود.
- APIهای `/api/stock-sleep/*` وجود داشتند.
- اما صفحه UI و route منوی «خواب کالا نماینده» به موتور جدید وصل نشده بود.
- بنابراین کاربر بعد از نصب، تغییر عملی در UI نمی‌دید.

## اصلاحات
- اضافه شدن منوی «خواب کالا نماینده» در بخش مدیریت.
- اضافه شدن صفحه اجرای Snapshot.
- دکمه آماده‌سازی Collectionها.
- دکمه ایجاد Snapshot تست با maxItems پیش‌فرض 10.
- دکمه پردازش Batch.
- نمایش وضعیت صف و درصد پیشرفت.
- نمایش Snapshotهای اخیر.
- نمایش خلاصه تأمین‌کننده/منشأ.
- نمایش لایه‌های کالایی منشأ انتخاب‌شده.

## مسیر تست پیشنهادی
1. ورود با اکانت admin.
2. منوی مدیریت → خواب کالا نماینده.
3. کلیک روی «آماده‌سازی Collectionها».
4. ایجاد Snapshot تست با 10 کالا.
5. چند بار «پردازش Batch بعدی».
6. Refresh وضعیت.

## محدودیت حفظ‌شده
- SQL مستقیم ندارد.
- محاسبه real-time سنگین ندارد.
- Sale Engine، Lead، Tablo، Search و Auto Sync تغییر نکرده‌اند.
