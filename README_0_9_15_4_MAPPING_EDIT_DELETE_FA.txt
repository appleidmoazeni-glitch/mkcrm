MKCRM 0.9.15.4 - اصلاح ویرایش/حذف اتصال صندوق و جاری کارکنان

اصلاحات:
- در بخش «تعریف صندوق/نماینده»، امکان ویرایش اتصال اشتباه اضافه شد.
- امکان پاک کردن جداگانه «صندوق فروشنده» اضافه شد.
- امکان پاک کردن جداگانه «جاری کارکنان / نماینده فروش» اضافه شد.
- امکان حذف کامل اتصال کاربر بدون حذف خود کاربر اضافه شد.
- جدول اتصال‌ها دارای دکمه‌های «ویرایش» و «حذف اتصال» شد.
- API حذف اتصال اضافه شد:
  DELETE /api/user-mappings?username=USERNAME
- ذخیره اتصال ناقص با تأیید ادمین مجاز است تا اتصال اشتباه قابل اصلاح باشد.

تست پیشنهادی:
1. با ادمین وارد شوید.
2. حسابداری/شایگان > تعریف صندوق/نماینده.
3. روی «ویرایش» کنار یک کاربر کلیک کنید.
4. صندوق یا جاری کارکنان را با دکمه پاک کردن خالی کنید و ذخیره کنید.
5. دوباره صندوق/جاری درست را سرچ و ذخیره کنید.
6. حذف کامل اتصال را تست کنید؛ کاربر باید باقی بماند، ولی در فاکتور فروش تا زمان تعریف اتصال جدید امکان صدور نداشته باشد.
