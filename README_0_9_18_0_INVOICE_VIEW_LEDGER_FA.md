# MKCRM 0.9.18.0 - Invoice View From Account Ledger

## اصلاحات

- در گردش حساب، شماره فاکتور clickable شد.
- با کلیک روی شماره فاکتور، فاکتور مستقیم از شایگان خوانده و به‌صورت read-only نمایش داده می‌شود.
- فروشنده فقط امکان مشاهده و چاپ دارد.
- مدیر و حسابداری می‌توانند یادداشت مدیریتی و اصلاح Lead ID در CRM را با ثبت علت انجام دهند.
- تغییرات مالی مثل کالا، قیمت، تعداد و تخفیف از CRM غیرفعال است و فقط باید در شایگان انجام شود.
- لاگ یادداشت‌ها و اصلاح Lead ID در Mongo ثبت می‌شود.

## حفظ اصلاحات قبلی

Lead soft sync، بانک مشتریان، گزارش سایت/کنسول، جستجوی فارسی/عربی، حذف رزرو شماره، شماره نهایی در لحظه صدور، HasDuplicateTrackingCode، فرمت سه‌رقمی و سرچ مشتری در فاکتور فروش حفظ شده‌اند.

## نصب

.env عملیاتی را overwrite نکن. خطوط Lead ID باید باقی بمانند:

```env
LEAD_ID_BASE_URL=http://127.0.0.1:8010
LEAD_ID_API_KEY=
LEAD_ID_TIMEOUT_MS=2500
```

بعد از نصب:

```bat
pm2 restart mkcrm --update-env
```

## تست

1. وارد گردش حساب صندوق فروشنده شو.
2. روی شماره فاکتور کلیک کن.
3. فاکتور باید از شایگان باز شود.
4. با فروشنده، فقط مشاهده و چاپ داشته باش.
5. با ادمین/حسابداری، یادداشت و اصلاح Lead ID با علت را تست کن.
