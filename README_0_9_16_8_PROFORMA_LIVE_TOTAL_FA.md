# MKCRM 0.9.16.8

## اصلاحات
- حفظ اصلاح جستجوی فارسی/عربی نسخه 0.9.16.7.
- اضافه شدن جمع زنده ردیف‌های پیش‌فاکتور هنگام افزودن، ویرایش یا حذف ردیف.
- نمایش جمع تعداد و جمع کل پایین جدول پیش‌فاکتور قبل از ثبت/چاپ.
- قفل ضد ثبت تکراری، اتصال Lead ID و سایر اصلاحات قبلی حفظ شده‌اند.

## تست
1. وارد پیش‌فاکتور شوید.
2. چند ردیف اضافه کنید.
3. پایین جدول باید جمع تعداد و جمع کل نمایش داده شود.
4. یک ردیف را ویرایش یا حذف کنید؛ جمع باید همان لحظه اصلاح شود.
