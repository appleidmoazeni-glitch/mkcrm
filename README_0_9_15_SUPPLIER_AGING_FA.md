# راهنمای تست نسخه 0.9.15

بعد از نصب، با ادمین وارد شوید و منوی «خواب کالا - تأمین‌کننده» را باز کنید.

## تست‌ها
1. دکمه «افزودن ۳ تأمین‌کننده پیش‌فرض» را بزنید.
2. باید این سه مورد دیده شوند: ماندگار، رایان کاوه، ماتریس.
3. روی «محاسبه گزارش» بزنید.
4. خروجی تأمین‌کننده‌محور و لایه‌های کالایی را بررسی کنید.
5. در بخش جستجو، یک تأمین‌کننده مثل «آونگ» یا «الماس» را جستجو و اضافه کنید.
6. برای تأمین‌کننده می‌توانید راس تسویه دستی وارد کنید.

## نکته مدیریتی
گزارش فعلی نسخه شروع است. برای تصمیم قطعی خرید باید خروجی ۳ تأمین‌کننده اول با واقعیت شایگان و شناخت بازرگانی مقایسه شود.
