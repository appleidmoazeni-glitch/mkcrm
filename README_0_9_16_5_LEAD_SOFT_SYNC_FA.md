# MKCRM 0.9.16.5 - Lead Soft Sync

## هدف

بعد از صدور موفق فاکتور فروش در شایگان، اگر Lead ID دستی در فاکتور وارد شده باشد، MKCRM اطلاعات فاکتور را به برنامه مستقل Lead ID ارسال می‌کند تا همان Lead با Closer واقعی بسته و قفل شود.

## رفتار مهم

- اگر Lead ID خالی باشد، فاکتور صادر می‌شود و مشمول گزارش کسر پورسانت می‌ماند.
- اگر Lead ID وارد شده باشد ولی برنامه Lead ID قطع باشد، فاکتور لغو نمی‌شود.
- خطای sync در `saleIssueLocks` و `leadSyncLogs` ثبت می‌شود.
- MKCRM هیچ دیتابیس Lead ID را مستقیم دستکاری نمی‌کند.

## متغیرهای `.env`

```env
LEAD_ID_BASE_URL=http://127.0.0.1:8010
LEAD_ID_API_KEY=
LEAD_ID_TIMEOUT_MS=2500
```

اگر Lead ID روی سرور دیگری است، IP همان سرور را بگذارید.
