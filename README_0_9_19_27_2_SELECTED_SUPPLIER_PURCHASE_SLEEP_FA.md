# MKCRM 0.9.19.27.2 - Selected Supplier Purchase Sleep

## هدف
مسیر خواب کالا تأمین‌کننده از حالت سنگین خواندن همه فاکتورهای خرید خارج شد.
در این نسخه ابتدا تأمین‌کننده با همان مسیر جستجوی حساب/گردش حساب انتخاب می‌شود؛ سپس فقط فاکتورهای خرید همان تأمین‌کننده از ابتدای سال خوانده و تحلیل می‌شود.

## تغییرات
- حذف نیاز به scan سنگین همه تأمین‌کنندگان در UI خواب کالا.
- اضافه شدن جستجوی تأمین‌کننده در صفحه خواب کالا تأمین‌کننده.
- خواندن فاکتورهای خرید تأمین‌کننده انتخاب‌شده از مسیر گردش حساب دقیق و سپس Invoice/Get.
- ساخت Snapshot فقط برای تأمین‌کننده انتخاب‌شده.
- حفظ شاخص رسوب بیش از ۳۰ روز و بیش از ۶۰ روز.
- حفظ Sale Guard و عدم تغییر مسیر صدور فروش.
- بدون SQL مستقیم.

## مسیر تست
1. مدیریت → خواب کالا تأمین‌کننده
2. آماده‌سازی
3. جستجوی تأمین‌کننده، مثلاً ماتریس
4. انتخاب تأمین‌کننده
5. خواندن فاکتورهای همین تأمین‌کننده
6. تحلیل خواب کالای همین تأمین‌کننده

## نکته تحلیلی
این Snapshot فقط برای تأمین‌کننده انتخاب‌شده است و برای کاهش فشار، full-market FIFO بین همه تأمین‌کنندگان را انجام نمی‌دهد. هدف این نسخه تست سریع و قابل کنترل برای هر تأمین‌کننده است.
