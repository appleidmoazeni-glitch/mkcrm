# MKCRM 0.9.19.42 - Sale Snapshot True Sale Type 2

## اصلاح کلیدی
نگاشت InvoiceType شایگان اصلاح شد:

- `InvTyp=2` = فروش واقعی
- `InvTyp=3` = خرید
- `InvTyp=6` = برگشت از فروش
