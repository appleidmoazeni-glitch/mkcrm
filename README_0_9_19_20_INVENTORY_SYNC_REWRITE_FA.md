# MKCRM 0.9.19.20 - Inventory Sync Rewrite

- بازنویسی وضعیت Auto Sync موجودی با زمان سرور.
- اجرای اولیه Auto Sync بعد از شروع سرویس با تأخیر کوتاه، سپس هر ۵ دقیقه.
- دکمه Sync Inventory Catalog همچنان موجودی مثبت تمام انبارهای فعال را جداگانه می‌خواند.
- Auto Sync Status و اجرای دستی یک tick به ابزارها اضافه شد.
- گردش حساب: اگر تاریخ وارد شود کل بازه نمایش داده می‌شود؛ اگر تاریخ خالی باشد فقط ۳۰ گردش آخر، جدیدترین اول.
- SQL مستقیم همچنان حذف است.
