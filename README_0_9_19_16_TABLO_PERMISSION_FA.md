# MKCRM 0.9.19.16 - اصلاح دسترسی فاکتور خرید در تابلو

- تابلو برای همه کاربران مجاز به ورود CRM فعال و قابل مشاهده است.
- بخش مرور ۳۰ فاکتور خرید آخر فقط برای ادمین یا کاربر دارای تیک `مرور فاکتورهای خرید` نمایش داده می‌شود.
- APIهای مرور فاکتور خرید نیز همین دسترسی را enforce می‌کنند.
- اعلام ورود کالا در تابلو فقط برای ادمین یا کاربر دارای تیک `اعلام ورود کالا در تابلو` مجاز است.
- مدیریت وضعیت رویدادهای تابلو فقط برای ادمین یا کاربر دارای تیک `مدیریت وضعیت تابلو` مجاز است.
- مشاهده خود تابلو و رویدادهای اتمام/ورود کالا برای همه کاربران دارای منوی تابلو باقی ماند.
