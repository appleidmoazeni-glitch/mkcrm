نسخه 0.9.19.2 - SQL Read Only Safe

مبنا: 0.9.19.1 که گزارش خواب کالا/تأمین‌کننده قابل قبول داشت.

معماری:
- SQL فقط برای خواندن موجودی، کالا، کاردکس، FIFO و لایه‌های خواب کالا استفاده می‌شود.
- گردش حساب/Turnover از WebService شایگان خوانده می‌شود.
- صدور و عملیات نوشتنی همچنان از WebService انجام می‌شود.
- اتصال SQL با کاربر read-only mkcrm_reader و دیتابیس CY000002 تنظیم شده است.
- حالت SHAYGAN_SQL_READONLY_STRICT=true جلوی استفاده از sa/admin را در مسیر SQL می‌گیرد.

هشدار نصب:
.env عملیاتی فعلی را بدون بررسی overwrite نکنید. اگر رمز mkcrm_reader را عوض کردید، مقدار SHAYGAN_SQL_PASSWORD را هم تغییر دهید.
